Authentication

All API requests require authentication using an API key. Include your API key in the Authorization header.

Request Headers 
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
Example Request 
curl https://api.blackroad.io/v1/inference \
  -H "Authorization: Bearer sk_live_abc123..." \
  -H "Content-Type: application/json" \
  -d '{
    "model": "lucidia-v2",
    "prompt": "Explain quantum computing",
    "max_tokens": 500
  }'

Core API Endpoints

POST /v1/inference

Generate AI-powered responses using BlackRoad's state-of-the-art models.

Parameters

Parameter Type Required Description
model string Required Model ID (e.g., "lucidia-v2", "quantum-sim")
prompt string Required Input prompt for the model
max_tokens integer Optional Maximum tokens to generate (default: 1000)
temperature float Optional Sampling temperature 0-1 (default: 0.7)
stream boolean Optional Enable streaming responses (default: false)
Example Response 
{
  "id": "inf_abc123",
  "model": "lucidia-v2",
  "choices": [{
    "text": "Quantum computing harnesses quantum mechanics...",
    "finish_reason": "stop",
    "index": 0
  }],
  "usage": {
    "prompt_tokens": 4,
    "completion_tokens": 156,
    "total_tokens": 160
  },
  "created": 1704123456
}

POST /v1/agents

Deploy autonomous AI agents for long-running tasks.

Parameters

Parameter Type Required Description
name string Required Agent name
model string Required Base model to use
tools array Optional Available tools for the agent
schedule string Optional Cron schedule for recurring tasks
Example Request 
{
  "name": "data-analyzer",
  "model": "lucidia-v2",
  "tools": ["web_search", "code_execution", "file_io"],
  "schedule": "0 */6 * * *",
  "config": {
    "max_iterations": 50,
    "timeout": 3600
  }
}
POST /v1/quantum/simulate

Run quantum circuit simulations with up to 32 qubits.

Example Request 
{
  "qubits": 4,
  "gates": [
    {"type": "H", "target": 0},
    {"type": "CNOT", "control": 0, "target": 1},
    {"type": "CNOT", "control": 1, "target": 2},
    {"type": "CNOT", "control": 2, "target": 3}
  ],
  "shots": 1000
}
Example Response 
{
  "job_id": "qsim_xyz789",
  "results": {
    "0000": 523,
    "1111": 477
  },
  "fidelity": 0.998,
  "execution_time_ms": 234
}

Error Codes

Code Status Description

unauthorized 401 Invalid or missing API key
rate_limit_exceeded 429 Too many requests
